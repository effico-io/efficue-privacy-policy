# EffiCue
Efficient Task Alarms and Cues.

Minimal UI | Modern Theme | Full Functional<br>

EffiCue is a local task and alarm manager for reminders, categories, custom sounds, and repeat scheduling.

## Privacy Policy
EffiCue stores alarms, task data, categories, and uploaded audio locally on the device. No user registration is required, and the app does not transmit personal data to external servers. Uploaded sounds are kept in app-private storage and used only for alarm playback.
