EffiCue

Efficient Task Alarms and Cues

EffiCue is a Flutter-based task and alarm manager for scheduling reminders, organizing categories, and playing custom alarm sounds.

What EffiCue does

  • Create tasks and associate alarms with each task.
  • Schedule alarms with exact timing and repeat patterns.
  • Choose custom audio files for alarms using local file picker uploads.
  • Store task, category, alarm, and sound data locally on the device.

App features

  • Task categories for organizing alarms and reminders.
  • Custom alarm audio uploads from the device.
  • Local SQLite storage for alarms, tasks, categories, and sound metadata.
  • Notification-based alarm delivery with exact alarm scheduling.

Permissions used

  • Notifications: required to show alarm alerts.
  • Exact alarm scheduling: required to trigger alarms at the selected time.
  • Audio/media file access: required only when the user uploads a custom alarm sound.

Privacy and data handling

EffiCue stores all user data locally on the device. No account or personal profile is required, and the app does not transmit your data to external servers.

Uploaded sounds are saved in app-local storage and used only for alarm playback.
